========================================
SIMPLE WORKFLOW EXPLANATION
========================================

When you run: python3 main.py

┌─────────────────────────────────────┐
│ 1. STARTUP                          │
│    - Load config.json               │
│    - Initialize all components      │
│    - Open cameras                   │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│ 2. MAIN LOOP (Runs 15 times/second) │
│                                     │
│    For each frame:                  │
│                                     │
│    A. CAPTURE                       │
│       └─> Read frame from camera    │
│                                     │
│    B. DETECT                        │
│       └─> Find face in frame        │
│       └─> Find eyes                 │
│       └─> Check if eyes open/closed │
│                                     │
│    C. CALCULATE                     │
│       └─> PERCLOS (eye closure %)   │
│       └─> Blinks per minute         │
│       └─> Risk Score (0-100)        │
│                                     │
│    D. CHECK ALERT                   │
│       └─> If score ≥ 50?            │
│       └─> Yes → Trigger buzzer      │
│       └─> No → Continue             │
│                                     │
│    E. DISPLAY                       │
│       └─> Draw HUD on frame         │
│       └─> Show metrics              │
│                                     │
│    F. LOG                           │
│       └─> Save to CSV file          │
│                                     │
│    G. REPEAT                        │
│       └─> Next frame...             │
│                                     │
└─────────────────────────────────────┘

KEY METRICS:

PERCLOS = How much eyes are closed (0.0 to 1.0)
  - 0.0 = Always open
  - 0.15 = 15% closed (threshold)
  - 0.30 = 30% closed (drowsy)

Blinks/min = How many blinks per minute
  - Normal: 15-20
  - Low: <10 (concerning)

Risk Score = Combined score (0 to 100)
  - 0-49 = Safe (no alert)
  - 50+ = Alert triggers

ALERT TRIGGERS WHEN:
  Risk Score ≥ 50

RISK SCORE = 
  (PERCLOS × 60%) + (Blinks × 40%) + Penalties

PENALTIES:
  +10 for yawn
  +5 for head nod
  +25 for texting
  +30 for car close

========================================
QUICK REFERENCE
========================================

Every Frame (15 FPS):
  → Capture image
  → Detect face/eyes
  → Update PERCLOS
  → Update blink count
  → Calculate score
  → Check alert
  → Display HUD
  → Log data

Every Second:
  → Write to CSV file

Every 5 Seconds:
  → Print status to console

Every 8 Seconds:
  → Can trigger new alert (cooldown)
