QUICK DEPLOYMENT:
-----------------
1. Copy entire "raspberry" folder to your Raspberry Pi

2. SSH into Pi:
   ssh pi@your-pi-ip

3. Navigate to folder:
   cd raspberry

4. Run installation:
   chmod +x INSTALL.sh
   ./INSTALL.sh

5. Configure cameras:
   - Connect USB cameras
   - Check: ls -l /dev/video*
   - Edit config.json with camera indices

6. Run:
   python3 main.py


CONFIGURATION:
--------------
Edit config.json to set:
- Camera indices (driver_cam_index, road_cam_index)
- GPIO pins (buzzer_gpio_pin, mute_button_gpio_pin)
- Alert threshold (risk_threshold)
- Resolution and FPS
